# Projects

## Open-Pit Mine Scheduling Optimizer
Large-scale MILP models and heuristic algorithms for open-pit mine production scheduling.

**Key features**
- mixed-integer programming formulation
- precedence constraints
- resource constraints
- cutting planes
- rolling horizon
- Large Neighborhood Search

## Relax-and-Repair Optimization Framework
A framework for obtaining feasible solutions quickly by relaxing selected hard constraints and repairing infeasibilities.
